Welcome to the four.one.nine blog! This is our first post on the new Astro static site.

## What's New

We've migrated from a complex Payload CMS + Next.js + Docker stack to a simple Astro static site with Markdown content.

### Benefits

- **Faster builds** - No more Docker containers
- **Simpler deployment** - Push to Git, auto-deploy via FTP
- **Static output** - Ultra-fast page loads
- **Markdown content** - Easy to edit locally
