In a role I once had for my day-job, we had to do a specialized type of work called transmission line loadability, which is a type of calculation for how much power transmission lines could handle safely. I developed calculatorium as a companion application when I wanted to learn React to handle many of the common calculations we did on a daily basis. I've since left that team, but they still use calculatorium to this day!
